# Blog Article Page Using Semantic HTML

## Documentation

---

## 1. Project Overview

This project is a *Blog Article Web Page* created using *HTML5 semantic elements*.  
The webpage explains the importance of web development and demonstrates how semantic tags are used to structure content clearly.

---

## 2. Purpose of the Project

- To learn the basic structure of an HTML document  
- To understand HTML5 semantic elements  
- To create a simple blog article layout  
- To improve content organization and readability  

---

## 3. Technologies Used

- HTML5

---

## 4. Description of the Web Page

The webpage contains the following main parts:

- Blog title and introduction  
- Main article content  
- Subtopics using sections  
- Author information  
- Footer with copyright  

---

## 5. Explanation of HTML Elements Used

### 5.1 `<!DOCTYPE html>`

Declares that the document is written in *HTML5*.

---

### 5.2 `<html lang="en">`

Specifies that the language of the webpage is *English*.

---

### 5.3 `<head>`

Contains page information such as:

- Character encoding (UTF-8)  
- Page title (The Web Learner)

---

### 5.4 `<header>`

Used twice in the page:

- Main header – Displays the blog title and tagline  
- Article header – Displays the article title and introduction  

---

### 5.5 `<article>`

- Contains the main blog content  
- Represents an independent blog article  

---

### 5.6 `<section>`

- Divides the article into subtopics  
- Used for:
  - Career Opportunities  
  - Creativity and Problem Solving  

---

### 5.7 `<aside>`

Contains author-related information, including:

- Author heading  
- Image  
- External link  
- Short description  

---

### 5.8 `<footer>`

- Displays copyright information  
- Appears at the bottom of the webpage  

---

## 6. Image and Link Usage

### Image

- Displays the author image using the <img> tag  
- The alt attribute improves accessibility  

### Link

- The <a> tag is used to create an external link
